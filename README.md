**StudyMate**

Studymate - is a web-based school management application, designed to organize and manage students, teachers, groups, announcements and schedule . 

This platform is replaces manual tracking with a structured digital solution where administrators can control group assignments, instructor information, course organization, and communication workflows from a single dashboard.

Studymate supports:
•	Maintaining detailed student and teacher profiles
•	Structuring learners into groups
•	Managing academic schedules and course assignments
•	Publishing institutional announcements
•	Importing bulk data through file uploads
•	Archiving and restoring records when necessary

The system is built with a focus on clarity, data consistency, and scalability, making it suitable for growing educational organizations that require reliable and organized management tools.

### **Main Features:**

**Group Management**
•	Create and manage study groups
•	Assign students to groups
•	Organize learning structure by cohorts

**Course Management**
•	Create courses
•	Assign teachers to courses
•	Link courses to groups

**Teacher Management**
•	View teacher list
•	Add new teachers
•	Edit teacher information
•	Delete teachers
•	Assign specialization (QA, SDET, etc.)

**Student Management**
•	View a list of all students
•	Search students by name/ email
•	Add new students
•	Edit existing student information
•	Delete students (move to Trash)
•	Assign students to groups
•	Specify study format (e.g., ONLINE)
•	Import students via Excel file

**Announcement Management**

•	creating announcement for students and teachers

•	specifying the group for announcement
**Trash Management**

•	restore deleted records

•	permanently delete records
•	deletion date
**Schedule Management**
•	Create events/schedule
•	Assign teachers to courses
•	Link courses to groups
